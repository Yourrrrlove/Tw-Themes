# Typora For Trwite Themes

## 使用

```text
git clone https://github.com/tree-white/Tw-Themes.git
cd Tw-Themes
npm install
npm run build
复制打包后 `dist` 里面的 `trwite` 文件夹和 `trwite.css` 到 `Typora` 主题文件中
```
